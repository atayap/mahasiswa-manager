# Mahasiswa Manager

Aplikasi manajemen data mahasiswa berbasis PHP + MySQL dengan fitur CRUD dan upload foto.

## Features
- ✅ Login Admin
- ✅ CRUD Data Mahasiswa
- ✅ Upload Foto
- ✅ Dashboard dengan Bootstrap
- ✅ Responsive Design

## Tech Stack
- PHP 7.4+
- MySQL
- Bootstrap 5
- PDO Database Connection

## Installation
1. Import database from `database.sql`
2. Configure database in `config/database.php`
3. Access via browser
